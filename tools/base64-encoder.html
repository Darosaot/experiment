About Base64

Base64 is a way of representing binary or text data using only 64 safe ASCII characters. It's everywhere in computing: embedding images in CSS or HTML (data URIs), encoding email attachments, storing tokens, and transmitting data through systems that only handle text.

Unicode-safe

Naive Base64 tools break on emoji and accented characters. This one encodes via UTF-8 first, so café and 🌍 round-trip perfectly.

FAQ

Is Base64 encryption?

No. Base64 is encoding, not encryption — anyone can decode it. Never use it to protect secrets.

Is my data uploaded?

No. Encoding and decoding happen entirely in your browser.
