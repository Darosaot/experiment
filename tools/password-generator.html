About this password generator

Reusing the same password across sites is the single biggest cause of account takeovers. This tool helps you create a unique, high-entropy password for every account. Passwords are generated locally using your browser's cryptographically secure random number generator (crypto.getRandomValues) — they are never transmitted, logged, or stored.

What makes a password strong?

  • Length matters most. Aim for at least 16 characters. Each extra character multiplies the time needed to crack it.
  • Mix character types. Combining upper-case, lower-case, numbers and symbols dramatically increases entropy.
  • Make it unique. A strong password reused everywhere is still risky. Use a different one per site, stored in a password manager.

FAQ

Are these passwords safe to use?

Yes. They're generated in your browser with a cryptographically secure random source and never leave your device. For maximum safety, generate the password on a device you trust and store it in a reputable password manager.

Should I include symbols?

Generally yes — symbols increase strength. If a particular site rejects certain symbols, untick the box and increase the length instead.

Do you store generated passwords?

No. Nothing is saved or sent anywhere. Refreshing the page erases everything.
