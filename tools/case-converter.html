About this case converter

Re-typing text just to fix its capitalization is a waste of time. This converter transforms whatever you paste into eight common cases instantly. It's handy for writers fixing accidental caps-lock, developers converting variable names, and marketers formatting headlines.

The cases explained

  • Title Case capitalises the first letter of each word — ideal for headlines.
  • Sentence case capitalises only the first letter of each sentence.
  • camelCase, snake_case and kebab-case are programming conventions for naming variables, files and URLs.

FAQ

Does it change the text in place?

Yes — the converted text replaces what's in the box, so you can keep converting or copy it directly.

Is my text uploaded?

No. All transformations run locally in JavaScript. Nothing is sent to a server.
