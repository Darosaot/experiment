About this JSON formatter

JSON (JavaScript Object Notation) is the standard format for APIs and config files, but it's often delivered as one unreadable line. This tool pretty-prints it with clean indentation so you can actually read it, validates that it's well-formed (pointing out where it breaks), and can minify it back down to save bytes.

Private by design

Many online JSON tools send your data to a server to format it — risky if that JSON contains tokens, keys or personal data. This one parses and formats entirely in your browser, so nothing ever leaves your machine.

FAQ

What does "Invalid JSON" mean?

Your text isn't valid JSON — common causes are trailing commas, single quotes instead of double quotes, or unquoted keys. The status line shows where parsing failed.

Does it handle large files?

Yes, within your browser's memory. Very large files depend on your device.
