About this color converter

Designers and developers constantly switch between color formats: a design tool gives you HEX, CSS wants RGB or HSL, and a brand guide lists something else entirely. This converter keeps all three formats in sync as you edit any one of them, with a live swatch so you always see exactly what you're working with.

The formats

  • HEX — six hex digits like #6c8cff, the most common web format.
  • RGB — red, green and blue from 0–255, e.g. rgb(108, 140, 255).
  • HSL — hue, saturation and lightness, intuitive for adjusting shades.

FAQ

Can I paste any format?

Yes — edit the HEX, RGB or HSL box directly and the others update. You can also use the color picker.

Does it support alpha/transparency?

This tool focuses on solid colors. For transparency, append an alpha value in your CSS, e.g. rgba(108,140,255,0.5).
